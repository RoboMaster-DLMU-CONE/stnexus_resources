# 11 CAN M3508

## 实验目标

通过 CAN 收发 M3508 电机与 C620 电调数据，解析转子角度、转速、电流和温度，并在明确的安全边界内发送控制电流。

## 上电前检查

- CANH/CANL 线序一致并可靠共地。
- 总线两端各有 120 Ω 终端，断电测量约为 60 Ω。
- 开发板保持 USB 5 V 供电，TJA1051 正常工作。
- 电机悬空或机械系统已解除危险负载，具备急停方式。

::: danger 先做只读验证
第一阶段只接收并解析反馈，不发送非零电流。确认电机 ID、反馈频率、角度跨圈处理和失联检测正确后，再从很小的限幅开始输出。
:::

## 软件结构

1. CAN 过滤器只接收需要的标准帧。
2. 中断回调读取帧并写入环形缓冲/队列。
3. 电机模块解析大端字段并更新时间戳。
4. 控制任务检查在线状态、限幅后统一发送电流帧。
5. 超时、急停或任务异常时发送零电流。

## 验收

- [ ] 手转电机时角度和转速反馈方向合理。
- [ ] 拔掉 CAN 后在规定时间内进入离线状态并清零输出。
- [ ] 控制电流有软硬限幅，启动不会突然满输出。
