# 05 PWM 蜂鸣器

## 实验目标

驱动 `PB9 / TIM4_CH4` 无源蜂鸣器，使用不同 PWM 频率产生音调，并实现一段不阻塞主循环的提示音序列。

## 关键概念

- 频率决定音调。
- 占空比影响驱动强度，建议从 20% 左右开始。
- 改变 `ARR` 时应同步更新 `CCR`，保持占空比合理。

## 非阻塞播放

将每个音符表示为“频率 + 持续时间”，周期调用播放器状态机：到期后切换到下一个音符，播放期间主循环仍可处理按键和传感器。

```c
typedef struct {
  uint16_t frequency_hz;
  uint16_t duration_ms;
} Note;
```

## 验收

- [ ] 能稳定输出至少三个可区分音调。
- [ ] 播放时按键仍可响应。
- [ ] 停止播放后 PWM 输出进入确定的静音状态。
