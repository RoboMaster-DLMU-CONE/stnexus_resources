# 02 GPIO 按键

## 实验目标

读取 `PA8` 与 `PA4` 两个按键，完成有效电平判断、消抖和按下边沿检测，并用 RGB LED 显示输入结果。

## 配置要点

1. 将两个引脚配置为数字输入。
2. 根据板卡版本的上拉电阻选择内部上拉/下拉或浮空输入。
3. 上电后先读取空闲电平，再定义按下有效值。

::: warning 先验证空闲电平
V1.2 说明书正文标注 SW3（PA4）带上拉电阻。不同修订或资料表格可能有出入，应以当前实物和原理图为准。
:::

## 推荐状态机

```text
空闲 -> 检测到有效电平 -> 等待 10~20 ms -> 再次确认 -> 产生一次按下事件
```

不要在按键保持期间持续触发事件。需要长按时，记录稳定按下时刻并在释放时计算持续时间。

## 验收

- [ ] 轻按一次只改变一次 LED 状态。
- [ ] 长按不会产生不可控的高速连发。
- [ ] 两个按键可以独立识别。
