# 08 SPI BMI270

## 实验目标

通过 SPI2 驱动 BMI270，读取三轴加速度和三轴角速度，完成静态零偏测量，并理解姿态数据进入控制系统前的处理流程。

## 硬件配置

| 信号 | 引脚 |
| --- | --- |
| CS | `PB12` |
| SCK | `PB13` |
| MISO | `PB14` |
| MOSI | `PB15` |

## 推荐调试顺序

1. 读芯片 ID，确认 SPI 模式、位序和 CS 时序正确。
2. 按官方驱动要求加载配置并检查返回状态。
3. 配置量程和输出数据率。
4. 静止读取原始值，确认重力轴约为 1 g。
5. 采样一段时间计算陀螺仪零偏。

::: danger 加热控制
IMU 加热电阻由 `PA1 / TIM2_CH2` 控制。基础读数实验无需开启加热；开发恒温控制时必须加入温度反馈、输出限幅和传感器故障保护，禁止持续 100% 占空比。
:::

## 验收

- [ ] 芯片 ID 稳定且符合器件手册。
- [ ] 转动板卡时对应轴数据方向一致、无随机跳变。
- [ ] 断开或读写失败时有错误码，不使用旧数据继续控制。
