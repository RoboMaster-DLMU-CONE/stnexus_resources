# 新生训练路线

这条路线以“每一阶段都能独立验证”为原则。建议先完成前一阶段的验收，再进入下一阶段。

## 阶段一：看得见的输入输出

- [ ] 让三色 LED 按固定顺序闪烁。
- [ ] 读取两个按键并完成软件消抖。
- [ ] 将电位器电压转换为 0～100% 的数值。

## 阶段二：定时器与波形

- [ ] 用 PWM 实现 LED 呼吸效果。
- [ ] 让无源蜂鸣器发出不同音调。
- [ ] 能说明频率、周期、占空比和计数器参数的关系。

## 阶段三：总线与设备驱动

- [ ] 读取 AHT20 温湿度数据并检查 CRC/状态。
- [ ] 在 OLED 上显示传感器数值与菜单。
- [ ] 读取 BMI270 加速度与角速度，理解坐标系和零偏。

## 阶段四：实时系统与机器人控制

- [ ] 将 LED、采样和显示拆分为 FreeRTOS 任务。
- [ ] 用队列或事件完成任务间通信。
- [ ] 通过 CAN 控制 M3508，并实现失联保护和输出限幅。
- [ ] 用 WS2812 显示系统状态，避免阻塞控制任务。

::: tip 验收建议
每次提交至少包含：配置说明、关键代码、运行现象、一个遇到的问题及其定位过程。训练的重点不是“例程能跑”，而是能解释为什么它能跑。
:::
