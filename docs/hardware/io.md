# LED、按键与 ADC

## 三色 LED

板载共阳极 RGB LED 的三个通道分别连接 `PB0`、`PA7`、`PA6`。GPIO 输出可用于状态指示，PWM 输出可用于调光、呼吸和颜色混合。

| 颜色 | 引脚 | 训练内容 |
| --- | --- | --- |
| 红 | `PB0` | GPIO 输出、状态告警 |
| 绿 | `PA7` | PWM 亮度、正常状态 |
| 蓝 | `PA6` | PWM 亮度、通信状态 |

## GPIO 按键

SW2 连接 `PA8`，SW3 连接 `PA4`。V1.2 说明书正文标注 SW3 带上拉电阻；不同板卡修订可能存在差异，配置前应通过原理图或万用表确认空闲电平。

按键程序至少处理三件事：有效电平、机械抖动和长按/短按判定。入门时可用 10～20 ms 延时消抖，进入实时系统后应改用周期扫描状态机。

## ADC 电位器

10 kΩ 电位器滑动端连接 `PA5 / ADC_IN5`。输入范围为 0～3.3 V，12 位 ADC 理论结果为 0～4095：

```text
voltage = adc_raw × 3.3 / 4095
percentage = adc_raw × 100 / 4095
```

::: tip 让读数更稳定
连续采样时可使用均值或中值滤波。先检查原始值是否覆盖合理范围，再添加滤波，避免用算法掩盖接线或 ADC 配置错误。
:::
