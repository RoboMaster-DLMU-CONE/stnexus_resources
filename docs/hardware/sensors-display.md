# 传感器与显示

## AHT20 温湿度传感器

AHT20 挂载在 I²C1，总线引脚为 `PB6/SCL` 和 `PB7/SDA`，7 位地址为 `0x38`。读取流程通常包括初始化检查、触发测量、等待完成、读取 6 字节数据并换算温湿度。

## 0.96 英寸 OLED

OLED 使用 SSD1306 驱动，与 AHT20 共用 `PB6/PB7`。说明书记录地址为 `0x78`；这是常见的 8 位写地址表示法，在使用 7 位地址的 API 中通常对应 `0x3C`。应根据驱动函数是否自动左移地址选择正确写法。

::: warning 共用 I²C 总线
AHT20 和 OLED 可以共存，但必须统一 I²C 句柄、时钟配置和总线访问策略。进入 FreeRTOS 后，多个任务访问同一总线时应使用互斥锁或集中式设备任务。
:::

## BMI270 六轴 IMU

BMI270 通过 SPI2 连接：

| 信号 | 引脚 |
| --- | --- |
| CS | `PB12` |
| SCK | `PB13` |
| MISO | `PB14` |
| MOSI | `PB15` |

驱动初始化应完成芯片 ID 检查、配置文件加载、量程和输出数据率设置。读取原始加速度与角速度后，还需要处理比例因子、坐标方向、静态零偏和滤波。

## 先做最小验证

- I²C：扫描或读取器件 ID/状态，不要一开始就写完整界面。
- OLED：先点亮一个像素或输出固定字符串。
- BMI270：先读芯片 ID，再读取静止原始值。
