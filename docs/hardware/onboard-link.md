# 板载 Link

开发板集成 Link 调试与烧录电路，一根 USB Type-C 数据线即可同时完成整板供电、程序下载和在线调试。相比外接调试器，这种连接方式更适合批量培训和快速定位问题。

## 可完成的工作

- 下载、擦除和校验固件。
- 设置硬件断点并单步执行。
- 查看寄存器、内存和实时变量。
- 配合 Ozone、J-Flash 等工具进行调试和烧录。

## 连接顺序

1. 先确认 BOOT 跳线处于正常启动位置。
2. 使用数据线连接电脑与开发板 Type-C 接口。
3. 等待系统识别调试设备。
4. 在工程中选择 STM32F103C8T6 与 SWD 接口。
5. 先执行一次连接/识别，再下载程序。

::: tip 保留恢复手段
修改时钟、低功耗或调试引脚前，先确保备用 SWD、复位和 BOOT 操作方式可用。调试连接失败时可尝试“复位下连接”，而不是立即判断硬件损坏。
:::
