# 继电器与扩展

## 板载继电器

板载继电器型号为 `SRD-05VDC-SL-C`，线圈通过 NPN 三极管驱动，控制引脚为 `PB5`。`PB5` 输出高电平时线圈吸合，常开触点闭合；输出低电平时释放。

| 触点 | 含义 |
| --- | --- |
| COM | 公共端 |
| NO | 常开端，吸合后与 COM 导通 |
| NC | 常闭端，释放时与 COM 导通 |

::: danger 负载边界
继电器触点额定值不是推荐工作值。本教程只建议在安全低压直流环境中学习，不提供市电接线指导。
:::

## 10P 扩展接口

开发板预留 10P 牛角座，用于连接 Stellaris-Nexus 扩展板或自定义模块。原说明书 V1.2 将详细定义标记为“待更新”，在正式使用前必须依据当前板卡版本的原理图逐针核对，不能根据排线方向猜测电源和信号。

## WS2812

WS2812 采用单线级联协议，对高低电平脉宽要求严格。V1.2 说明书中该部分仍待更新；编写驱动前先确认实际数据引脚、电压电平和供电能力，再选择定时器 PWM + DMA 或经过验证的软件驱动方案。
