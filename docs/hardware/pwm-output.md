# PWM 与执行器

## 舵机接口

舵机信号由 `PB8 / TIM4_CH3` 输出。常见位置舵机使用 50 Hz（20 ms 周期）控制脉冲，脉宽约 0.5～2.5 ms 对应其角度范围，具体数值必须以舵机规格书为准。

::: danger 先核对线序
3Pin 接口包含信号、5 V 和 GND。反接电源可能损坏舵机；大电流舵机还应使用独立电源，并与开发板共地。
:::

## 无源蜂鸣器

蜂鸣器由 `PB9 / TIM4_CH4` 的 PWM 驱动。频率决定音调，占空比影响驱动强度，说明书建议使用约 20% 占空比平衡响度与功耗。

## IMU 加热输出

`PA1 / TIM2_CH2` 控制 BMI270 附近的加热电阻。其目标是通过闭环控制让 IMU 保持稳定温度，从而减少零偏随温度变化。

::: warning 不要开环满功率加热
必须读取温度、限制最大占空比并处理传感器失效。禁止持续 100% 占空比输出；调试 PID 时从低输出上限开始。
:::

## PWM 参数关系

```text
PWM 频率 = 定时器时钟 / ((PSC + 1) × (ARR + 1))
占空比 ≈ CCR / (ARR + 1)
```

先选择便于计算的计数频率，再配置周期 `ARR` 和比较值 `CCR`。
