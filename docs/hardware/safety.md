# 供电与安全

## 上电检查清单

- [ ] 确认 Type-C 电源为 5 V，线缆和接口无破损。
- [ ] 外接模块的电压等级与开发板兼容，所有通信设备可靠共地。
- [ ] 舵机、CAN、SBUS 和继电器线序与标识一致。
- [ ] 板上无金属碎屑、焊锡或导线造成短路。
- [ ] BOOT 跳线处于预期位置，外部负载可以随时断电。

::: danger 发现异常立即断电
上电或运行中出现打火、冒烟、焦味、异常发热或器件异响，应立即切断电源，不要继续尝试烧录或复位。
:::

## 必须注意的接口

### CAN

CAN 收发器需要 USB 提供的 5 V 电源。接线前确认 `CANH` 对 `CANH`、`CANL` 对 `CANL`；整条总线断电后测量 CANH 与 CANL 之间的总电阻，正常双终端网络应约为 **60 Ω**。开发板本身没有终端电阻。

### 舵机

3Pin 接口线序为信号、5 V、GND。不同品牌线色不完全一致，必须按引脚定义核对；电源与地反接可能直接损坏舵机。

### BMI270 加热

加热电阻由 `PA1 / TIM2_CH2` 的 PWM 控制。仅在具备温度反馈与限幅时启用，**不得持续输出 100% 占空比**。

### 继电器

继电器触点与 MCU 控制侧电气隔离并不代表可以忽略高压安全。本训练站只讨论低压直流负载；涉及市电必须由具备资质和防护条件的人员完成。

## 静电与机械防护

避免直接触摸器件引脚、测试点和裸露焊盘。插拔扩展模块前先断电，连接器不要斜插或强行反向插入。
